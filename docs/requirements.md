# 讲座管理系统 - 需求文档

## 1. 项目概述

### 1.1 项目背景
讲座管理系统是一个用于管理和安排讲座活动的后台管理工具，支持配置讲座时间、讲师、学科，以及选择目标学生群体。

### 1.2 目标用户
- 教务管理人员
- 讲座协调员
- 机构管理员

---

## 2. 功能需求

### 2.1 讲座管理

#### 2.1.1 新建讲座
- **学科选择**：从预设学科列表中选择（数学、语文、英语、物理、化学）
- **讲师选择**：
  - 从内部老师列表选择（根据学科自动过滤）
  - 通过员工工号手动添加内部老师
  - 选择外部专家（带"外部"标签标识）
- **时间配置**：
  - 选择讲座日期（日期选择器）
  - 选择开始和结束时间（时间选择器，5分钟间隔）
- **目标学生**：支持多维度多选
  - 按机构选择（选择整个机构的所有学生）
  - 按班级选择（选择整个班级的学生）
  - 按学生名单选择（搜索并选择单个学生）

#### 2.1.2 编辑讲座
- 修改已创建讲座的所有配置项
- 点击讲座卡片即可进入编辑

#### 2.1.3 删除讲座
- 支持删除单个讲座
- 删除前需二次确认

#### 2.1.4 讲座展示
- 周视图时间轴布局（8:00-20:00）
- 按学科颜色区分讲座
- 显示讲座信息：学科名称、讲师、时间、学生人数
- **支持同一时间段多场讲座并排显示**
- **表头显示星期和具体日期（如 周一 03/10）**

#### 2.1.5 周切换
- 支持切换查看不同周的讲座
- 提供上一周/下一周切换按钮
- 显示当前周的日期范围（如 3月10日 - 3月14日）
- 提供"回到本周"快捷按钮
- 默认显示当前周的讲座

### 2.2 外部专家管理

#### 2.2.1 专家入驻
- 输入专家姓名
- 输入手机号码（需验证格式）
- 选择擅长学科（可多选，可不选）

#### 2.2.2 专家列表
- 显示所有外部专家
- 支持按姓名/手机号搜索
- 显示专家状态（正常/已禁用）

#### 2.2.3 专家管理
- 禁用专家账号（禁用后无法被选择安排讲座）
- 启用已禁用的专家
- 删除专家（需二次确认）

### 2.3 数据统计
- 本周讲座数量
- 授课教师数量
- 学科数量
- 外部专家数量

---

## 3. 数据结构

### 3.1 组织架构
```
机构
└── 班级
    └── 学生
```

### 3.2 人员类型
- **内部老师**：有员工工号，关联擅长学科
- **外部专家**：有手机号，可设置擅长学科，有状态控制

### 3.3 讲座数据
- 关联学科、讲师
- 时间信息（日期、星期、开始时间、结束时间）
- 目标学生（机构ID列表、班级ID列表、学生ID列表）

---

## 4. 界面要求

### 4.1 主界面
- 顶部导航栏：系统标题、功能按钮
- 周切换器：上一周/下一周按钮、日期范围显示、回到本周按钮
- 统计卡片区（显示当前选中周的数据）
- 周视图讲座表

### 4.2 交互要求
- 点击空白区域快速添加讲座
- 点击讲座卡片编辑
- 悬停显示操作按钮
- 表单验证提示

---

## 5. 非功能需求

### 5.1 兼容性
- 支持主流现代浏览器（Chrome、Firefox、Safari、Edge）
- 桌面端优先

### 5.2 响应速度
- 页面加载时间 < 3秒
- 操作响应时间 < 500ms

### 5.3 数据
- 当前为静态Demo，数据存储在前端状态
- 刷新页面数据重置为初始状态
